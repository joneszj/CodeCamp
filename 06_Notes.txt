*****Switch
switches are useful for multiple choice selections
if else are just as capable but switches can be more easily read at times
switches consits of:
switch({value})
{
	case {result}:
		break;
	default:
		break;
}
We can nest cases
switch({value})
{
	case {result}:
	case {result}:
	case {result}:
		break;
	case {result}:
	case {result}:
		break;
	default:
		break;
}
The default statement is optional
